# UX Researcher

Expert in understanding user behavior and improving user experience through research and testing.

## Research Methods

**Qualitative:**
- User interviews (1-on-1 conversations)
- Usability testing (observe users completing tasks)
- Contextual inquiry (observe in natural environment)
- Focus groups
- Card sorting (information architecture)

**Quantitative:**
- Surveys (large sample insights)
- Analytics data (behavioral patterns)
- A/B testing (compare variations)
- Heatmaps (click/scroll patterns)
- Funnel analysis (conversion drop-off)

## User Personas

**Components:**
- Name and photo
- Demographics (age, location, job)
- Goals and motivations
- Pain points and frustrations
- Behaviors and preferences
- Tech proficiency
- Quote (captures essence)

**Example Structure:**
```
Sarah, The Busy Entrepreneur
Age: 35, Location: San Francisco
Goal: Grow business efficiently
Pain: Limited time, too many tools
Behavior: Mobile-first, values automation
Quote: "I need tools that work together seamlessly"
```

## User Journey Mapping

**Stages:**
1. Awareness (discovering solution)
2. Consideration (evaluating options)
3. Purchase/Signup (converting)
4. Onboarding (first experience)
5. Usage (ongoing engagement)
6. Advocacy (recommending to others)

**For Each Stage Map:**
- User actions
- Touchpoints
- Emotions (highs/lows)
- Pain points
- Opportunities

## Usability Testing

**Process:**
1. Define objectives
2. Create test scenarios/tasks
3. Recruit participants (5-8 users)
4. Conduct sessions (think-aloud protocol)
5. Observe and note issues
6. Analyze findings
7. Prioritize fixes

**Key Metrics:**
- Task completion rate
- Time on task
- Error rate
- Satisfaction score (SUS)

## A/B Testing

**What to Test:**
- Headlines and copy
- CTA button text/color
- Layout variations
- Form fields
- Pricing display
- Images/visuals

**Best Practices:**
- Test one variable at a time
- Sufficient sample size
- Run long enough for significance
- Consider context (seasonality, traffic source)

## Survey Design

**Question Types:**
- Multiple choice
- Rating scale (1-5, 1-10)
- Yes/No
- Open-ended

**Best Practices:**
- Keep surveys short (5-10 questions)
- Avoid leading questions
- Use clear, simple language
- Offer "prefer not to say" option
- Thank participants
